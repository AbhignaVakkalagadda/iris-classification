# Iris Flower Classification Project

![Iris Flowers](https://upload.wikimedia.org/wikipedia/commons/thumb/4/41/Iris_versicolor_3.jpg/320px-Iris_versicolor_3.jpg)

## Project Overview
This machine learning project classifies Iris flowers into three species based on their sepal and petal measurements. The implementation includes data exploration, feature analysis, model development, and comprehensive evaluation.

## Dataset
The Iris dataset contains 150 samples with 4 features:
- Sepal length (cm)
- Sepal width (cm) 
- Petal length (cm)
- Petal width (cm)

Three target species:
- Iris-setosa
- Iris-versicolor  
- Iris-virginica

## Key Features
- Exploratory Data Analysis (EDA)
- Feature importance analysis
- Multiple model comparison
- Performance visualization
- Model persistence
